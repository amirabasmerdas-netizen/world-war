# ربات استراتژیک مولتی‌پلیر تلگرام

یک بازی استراتژیک چندنفره با ربات مادر و فرزند، سیستم اقتصادی، جنگ و هوش مصنوعی پویا.

## ویژگی‌ها

### 1. ربات مادر
- ثبت و مدیریت ربات‌های فرزند
- دریافت توکن و آیدی مالک
- نمایش لیست ربات‌ها

### 2. ربات فرزند
- مدیریت کشور و نیروها
- سیستم اقتصادی با وام
- جنگ با سایر کشورها و AI
- پنل مالک برای مدیریت کاربران

### 3. نیروها و سازه‌ها
- نیروی زمینی، هوایی، دریایی
- نیروی سایبری و تسلیحات ویژه
- کارخانه‌ها، معادن، نیروگاه‌ها
- سازه‌های خدماتی

### 4. هوش مصنوعی پویا
- کشورهای AI با شخصیت‌های مختلف
- تصمیم‌گیری استراتژیک هر ۱۰-۳۰ دقیقه
- قابلیت حمله، دفاع و تشکیل اتحاد

### 5. سیستم جنگ
- محاسبه خودکار نتایج جنگ
- وابسته به نیروها، تکنولوژی و شانس
- سرقت منابع از کشور مغلوب

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.13
- PostgreSQL (برای Render)
- توکن ربات تلگرام

### نصب روی Render

1. **فورک کردن ریپازیتوری**
2. **ایجاد اپلیکیشن جدید روی Render**
3. **وارد کردن متغیرهای محیطی:**
   - `MOTHER_BOT_TOKEN`: توکن ربات مادر
   - `DATABASE_URL`: آدرس دیتابیس PostgreSQL
   - `WEBHOOK_URL`: آدرس وب‌هوک

4. **استقرار خودکار**

### نصب محلی

```bash
# کلون کردن پروژه
git clone [repository-url]
cd strategic-bot

# نصب وابستگی‌ها
pip install -r requirements.txt

# تنظیم متغیرهای محیطی
export MOTHER_BOT_TOKEN="your_token"
export DATABASE_URL="sqlite:///game.db"

# راه‌اندازی ربات مادر
python main.py

# راه‌اندازی سرور وب‌هوک
python webhook_server.py
